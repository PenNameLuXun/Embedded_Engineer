# 第 22 章自检题答案

1. 1920×1080×24bit×60 = 2.98 Gbps（裸像素带宽）。加 ~20% DSI 包头 → 实际 ~3.6 Gbps。  
   4 lane × 1.5 Gbps = 6 Gbps 理论。考虑 ~80% 有效带宽 = 4.8 Gbps，**够**。  
   单 lane 1.5 Gbps 不够（即使 100% 效率也低于裸带宽）。

2. 多摄像头共享一条 CSI-2 链路（如某些汽车 ADAS 用一根 link 接 2-4 路摄像头）；同一摄像头同时输出多种格式（RAW + 缩略图）；HDR 多曝光帧标识。VC 让接收端能分流。

3. (1) Lane 同步：发送和接收的"位边界对齐"需要一段稳定的过渡。  
   (2) 时钟训练：CLK lane 切到 HS 时需要给 RX SerDes 锁相环时间。  
   (3) 让接收方从超低功耗状态唤醒，模拟前端预热。

4. **静态画面 + 极低功耗**场景。例如智能手表的 always-on display：屏幕显示时间，SoC 进入深度睡眠几秒钟才唤醒一次更新。Video Mode 需要 SoC display controller 持续供应像素，无法深睡。
