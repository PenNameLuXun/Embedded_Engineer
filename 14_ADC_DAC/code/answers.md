# 第 14 章自检题答案

1. 工频 50 Hz 信号本身只有几次谐波 (250 Hz 以下足够)，按 Nyquist 至少 500 Hz 采样。实际工程会选 1–10 kHz 留余量。抗混叠滤波必须加：环境噪声里 1 kHz / 10 kHz 工业干扰会被混叠到 50 Hz 附近，让你"看到"不存在的工频抖动。

2. 12 × 6.02 + 1.76 ≈ 74 dB。实际 ENOB 受四类因素拉低：(a) 热噪声 + 输入缓冲非理想；(b) 时钟抖动 → 转换时刻不准；(c) INL / DNL 非线性；(d) PCB 串扰 + 电源噪声。

3. 音频 (48 kHz, 16~24 bit) 对 SNR 要求 > 96 dB，SAR 难做到；Σ-Δ 用过采样把信噪比叠加上去 + 噪声整形抑制带内噪声，天生适合"带宽窄但分辨率高"的场景。高精度温度同理：变化慢、但要 16 位以上分辨率。

4. **RC 时间常数**（决定平滑度）+ **PWM 频率**（决定纹波频率）。要平滑 → RC 大。要响应快 → RC 小。trade-off 经典。
