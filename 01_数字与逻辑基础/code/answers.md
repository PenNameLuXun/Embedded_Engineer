# 第 01 章 自检题参考答案

## 1. `0x3F & ~0x0F` = ?

`0x3F = 0011_1111`，`~0x0F = ...1111_0000`（高位都是 1，但 `& 0x3F` 后只看低 8 位）：

```
0011_1111
1111_0000
---------
0011_0000   = 0x30
```

## 2. `(uint8_t)(0xFF + 1)` = ?

= `0`。  
8 位无符号溢出按定义"模 256"，是 well-defined 行为。**有符号**溢出才是 UB。

## 3. 把 `x` 第 12 位和第 17 位都置 1

```c
x |= (1u << 12) | (1u << 17);
```

## 4. 为什么 D 触发器需要时钟而 AND 门不需要？

AND 门是 **组合逻辑** —— 输出只依赖当前输入，没有记忆，没有"什么时候采样"的问题。  
D 触发器是 **时序逻辑** —— 它要"在某个时刻把输入锁存到输出"，那个时刻就是时钟边沿。没有时钟，它就不知道什么时候去采样、什么时候保持。

## 5. 8 位行波进位加法器的最坏延迟？

每位 1 ns，进位从最低位传到最高位需要经过 8 级全加器 → **8 ns**。  
这就是为什么高速加法器要用 carry-lookahead / carry-select 把"等进位"这个串行依赖打散。

## 6. 32 位大小端互换的一行

```c
uint32_t le = __builtin_bswap32(be);
```

或者纯 C：

```c
uint32_t le = ((be>>24) & 0xFF)       |
              ((be>>8)  & 0xFF00)     |
              ((be<<8)  & 0xFF0000)   |
              ((be<<24) & 0xFF000000);
```
