# 第 23 章自检题答案

1. **BLE 为首选**。门锁需要手机解锁（BLE 普及度最高 + 与 iOS/Android 兼容好）+ 电池供电几个月（BLE 待机极低功耗）。距离 < 10 m 完全够。Wi-Fi 太费电；LoRa 距离过剩 + 没有手机原生支持。  
   工业级智能锁会同时带 BLE + Zigbee 二选一，BLE 给手机，Zigbee 给家庭网关。

2. 不建立。iBeacon 只发广播包 (ADV_NONCONN_IND)，把 UUID + Major + Minor 编进 manufacturer data。手机扫描时直接读广播。**优点**：无连接开销，电池能用几年；同一信标能被多个手机同时检测。

3. KRACK 利用 WPA2 四次握手第 3 步的重放：攻击者强制 client 重新装载已用过的随机数 (nonce)，让相同 key + 不同明文产生 keystream 复用 → 推出明文。**根因**：协议没考虑重放对 nonce 计数器的影响。WPA3 SAE 用前向安全协议根治。

4. **多径衰落 + 障碍物吸收**。城市环境信号经墙壁、车辆、其它电磁源多次反射，到达接收机时多个相位的副本叠加 → 衰落极快。+ 2.4 GHz / 1 GHz 频段穿透混凝土的衰减 ≈ 6–10 dB / 墙，几堵就把-148 dBm 的链路预算用完。LoRa 在城市需要在屋顶部署 gateway 才能覆盖几公里。
