# 第 10 章 自检题答案

1. PL011 规定 `UARTEN=1` 时改 IBRD/FBRD/LCRH 行为未定义。必须先清 `UARTEN`，写完寄存器、写最后一笔 LCRH（这一笔触发寄存器从影子拷到工作寄存器），再开 `UARTEN`。

2. 中断撕裂 / read-modify-write 的非原子性。普通做法是：`GPIO_DATA |= (1u<<3)` = LDR/ORR/STR 三条指令，被中断打断时另一段代码改了同一寄存器 → 写回旧值覆盖。LM3S 的"地址即掩码"让 GPIO 操作变成单条 STR，**硬件原子**。

3. brd_x64 = (16e6 * 4) / 115200 = 555。 IBRD = 555 / 64 = 8，FBRD = 555 % 64 = 43。

4. 写这些寄存器的硬件副作用比"波特率本身"广 —— 比如 LCRH 那笔同时触发寄存器锁存，UARTEN 那笔解除复位。养成完整初始化的习惯，迁到真硬件时不用重写。

5. 写过去什么都不会发生：外设根本没被时钟驱动，总线握手会超时 → BusFault。这是新手常见 bug，第 11 章会顺带讲。
