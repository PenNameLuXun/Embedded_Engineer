# 第 12 章自检题答案

1. SysTick 是 24 位向下计数器，最大 LOAD = 2²⁴ − 1 = 16,777,215。  
   50 MHz CPU 时钟下：(16,777,215 + 1) / 50e6 ≈ 0.3355 秒 ≈ 335.5 ms。

2. 无符号减法在 ARM 上有"自动回绕"特性。`g_ticks - start` 算出的"时间差"在 `g_ticks` 32 位环回时仍然正确。  
   `g_ticks < start + ms` 如果 `start + ms` 溢出会算错，且 `g_ticks` 回绕到 0 时立刻满足条件，提前结束 delay。

3. 人眼能感觉 100 Hz 以下的闪烁。低于 60 Hz 明显闪、低于 30 Hz 像频闪灯。对电机驱动则会引入可听噪音（PWM 在音频带内）。所以电机驱动常用 20 kHz+ 频率。

4. **测短时间差用 DWT->CYCCNT**：分辨率 1 周期，没有中断进入开销。  
   **测长时间或 OS 任务间隔用 SysTick**：避免 CYCCNT 32 位回绕 (50 MHz 下 ~86 秒就绕一圈)。  
   中断响应延迟典型 < 1 µs，必须用 DWT。
