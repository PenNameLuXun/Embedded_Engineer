# 第 17 章自检题答案

1. 多设备共享一对线：(a) 任何设备能拉低；(b) 任何设备能释放（让线被上拉回高）；(c) 多个设备同时驱动不会"打架"烧 IO。推挽就会冲突 → 大电流 → 烧。开漏 = wired-AND 是物理上唯一安全的方案。

2. 主 B 赢。两人前 5 位一致 (01000)，第 6 位 A 想发 1，B 想发 0。SDA 实际是 0 (B 拉低)。A 看到自己想发 1 但线是 0 → 仲裁失败，退出。B 继续。**地址小者赢**。

3. 从机在时钟拉伸 —— 一直拉低 SCL 没释放，主以为"高电平没到"而等。或者从机挂死了把 SDA 也按住。工业系统会做"硬件超时复位"或主动产生 9 个 STOP+START 解锁总线。

4. 命令字节格式（每个事务带一个 cmd 字节）+ 通信超时（25 ms）+ PEC（包错误校验，可选）。SMBus 是 I²C 的 **应用层规范**。
