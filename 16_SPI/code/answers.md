# 第 16 章自检题答案

1. CPOL 不同：Mode 0 SCK 空闲低，上升沿是"采样沿（第一个沿）"；Mode 3 SCK 空闲高，上升沿是"采样沿（第二个沿）"。CPHA 不同导致主和从在数据"换边"的时刻不同。

2. 典型上限 25 MHz（标准 SD 卡 SPI 模式）。SD 总线模式（4 线数据 + 命令）可跑到 50 MHz 甚至 200 MHz（UHS-I/II）。SPI 慢的原因：(a) 单 bit 串行，(b) 单从机指令-响应往返开销，(c) SD 卡 SPI 模式不支持高速时钟切换。

3. 菊花链的本质是把所有从机的"内部移位寄存器"接成一个长串。从机 IP 必须在 nCS 低期间把"上一次接收到的字节"在下一个时钟周期发出去，从 MISO 出 → 形成接力。这是硬件行为，软件改不了。

4. 取决于：(a) QSPI 时钟频率（80–200 MHz），(b) 是否启用 cache。无 cache 时 QSPI XIP 取指延迟 ~30–100 ns；内部 Flash 一般 0~2 等待 ≈ 5–25 ns。所以**慢 2–10 倍**，但启用指令 cache 后命中率高 → 实际平均损失 < 2 倍。
