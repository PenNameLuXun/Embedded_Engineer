# 第 43 章自检题答案

1. **不等价，但近似**。int8 仅 256 个离散值，丢掉浮点的细微差异。精度只掉一点是因为：(a) 神经网络对小扰动鲁棒（训练时本身就有 dropout、SGD 噪声）；(b) 量化是 per-channel 的，每个通道独立校准 scale；(c) QAT 在训练里就模拟量化误差，让模型适应。但极端任务（GAN、Bayesian inference）量化会爆。

2. 关键词唤醒模型超小（< 20 KB）：(a) 任务限定（"Hey X"，二分类）；(b) 输入小（Mel-filterbank 40×40）；(c) 模型 5-7 层卷积；(d) int8 量化。MCU 持续监听 → 麦克风采样 → 16 ms 一帧推理 → 总功耗 100 µA 级，纽扣电池一年。

3. 取决于模型大小和 Cortex-M4 频率。168 MHz Cortex-M4 + CMSIS-NN 跑 MobileNetV1 0.25× (~200 K 参数)：**~100-200 ms / 推理**。1 fps 性能，做物体检测勉强能用。要实时需要 M7 / M55 + NPU。

4. **边缘优势**：(a) 隐私（数据不出设备）；(b) 延迟 < 100 ms；(c) 离线可用；(d) 带宽省。**劣势**：(a) 模型容量受限；(b) 升级慢（OTA 而不是云端热更新）；(c) 训练数据来源有限。**混合是最常态**：唤醒在边缘 + 复杂识别在云。
