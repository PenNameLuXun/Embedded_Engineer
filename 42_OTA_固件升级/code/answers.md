# 第 42 章自检题答案

1. (a) 关键模块自检（UART/Wi-Fi/传感器都通）；(b) 应用层 ping 自家服务器 / 收到心跳；(c) 一段固定时间（如 5 分钟）内没 reset；满足后才 boot_set_confirmed()。这套机制叫 **watchdog rollback**。

2. 差分需要旧版本作为基础 → 旧版本必须可信 + 完整。如果旧版本被部分覆盖了再失败 → 回滚目标已损坏。解决：把差分应用到 B 分区生成完整新镜像，再切换。**任何时候至少一个完整可用镜像**。

3. (a) Flash 可以擦写复位 → 攻击者擦了 Flash 让计数器回 0 即可装旧版；(b) OTP / e-Fuse 是**单向**的 (只能从 0 烧到 1)；(c) e-Fuse 在 die 内部，外部探针不易接触。

4. 攻击者**能逆向**固件（找漏洞、提取算法、了解协议）但**不能修改**（签名验失败）。所以未加密不影响升级安全性，但损害知识产权 + 可能暴露利用路径。商业产品通常加密 + 签名双重。
