# 第 15 章自检题答案

1. 9600 baud, 8N1 = 10 位 / 字节, 因此 960 字节/秒。10 KB ≈ 10240 字节 → 约 10.7 秒。

2. 1× 等于"每位采一次"，恰好在中间能采到就行，但波特率有偏差或抖动时容易采到边沿过渡 → 错。16× 采样能：(a) 找到起始位下降沿精确位置 → 后续位采样对齐到中点；(b) 容忍 ±1/16 波特率偏差；(c) 三次投票滤毛刺。4× 也有人用，省一些硬件但容差小。

3. 物理层：波特率不匹配、线驱动太弱被噪声压住、参考地不共、信号过冲穿透禁区。逻辑层：发送方 stop 位数和接收方设置不一致。

4. 至少 4 个：`IDLE`（等下降沿）、`START`（等到位中点确认起始）、`DATA`（采 8 位）、`STOP`（验证停止位）。工业实现还要 `ERROR` 状态做恢复。
