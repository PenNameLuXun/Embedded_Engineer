# 第 41 章自检题答案

1. P = C V² f。比例 = (1.8² × 100) / (0.9² × 50) = (3.24 × 100) / (0.81 × 50) = 324 / 40.5 = **8×**。即降一半电压 + 一半频率，动态功耗变 1/8。这就是 DVFS 收益的物理基础。

2. **时钟门控**：CPU / 模块还有电源，时钟被关 → 动态功耗 ~0，但漏电仍在；状态保持完整；唤醒极快（重开时钟）。**电源门控**：直接切断这块的 VDD → 动态 + 静态都为 0，但状态全丢；唤醒慢（重新初始化），有 in-rush current 冲击。深度睡眠用电源门控，浅睡眠用时钟门控。

3. **稀疏事件 + 任务周期不严格** 的场景，如 IoT 节点 (传感器每 1 小时上报一次)。原本 1 kHz tick 每秒唤醒 1000 次，tickless 把它压到几次 → 平均功耗下降 100×。**周期密集型 (10 kHz 控制环)** 用 tickless 帮助有限。

4. 浮空输入电平徘徊在 VIL/VIH 之间禁区，CMOS 输入级里 PMOS 和 NMOS 同时导通 → **通路短路电流 (shoot-through)**。每个浮空 GPIO 几十 µA 到 mA 不等。所以 datasheet 一律要求未用 GPIO 配置成 analog input 或加上 / 下拉。
