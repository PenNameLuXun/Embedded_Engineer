# 第 18 章自检题答案

1. 差分：抵抗共模电磁干扰（汽车里有强电机和点火噪声）。双绞线进一步降低环路面积，减少电磁耦合。这是 CAN 在恶劣电磁环境下仍能跑 km 的根本。

2. ID=0x000 是 11 位全 0，理论上最高优先级，但**实践中保留**或预先分配给极少数关键报文（同步 / 时间戳）。汽车里几个核心 ECU 共识 ID 表后才能用。

3. 仲裁段的电气线与要求：一个节点发的"显性"必须在**1 个位时间内**传到总线另一端并被所有节点采到，否则仲裁失败。1 Mbps 下 1 位 = 1 µs，对应电磁波在双绞线里约 200 m → 来回 + 收发器延迟 + 采样裕度 ≈ 40 m 上限。

4. CAN 控制器把节点强制下线（CAN_TX 拉到隐性，不再发送）。恢复条件由协议规定：监听到 **128 次 11 位连续隐性** (≈ 总线空闲一段时间) 后自动重置 TEC=REC=0 重新上线。软件也可以触发硬件复位（适用于硬故障已修好）。
